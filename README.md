# DFHack Twitch Integration

## Project Scope

DFHack Twitch Integration is a DFHack plugin package that connects Twitch chat to fortress interaction in Dwarf Fortress.

## Functionality

- Receives Twitch chat messages and routes supported commands to DFHack Lua handlers.
- Supports viewer claim/status actions and fortress interaction commands.
- Supports religion, squad, leave, and utility command groups.
- Provides OAuth/auth helpers, diagnostics, and runtime debug utilities.

## Package Contents

- `hack/plugins/dfxtwitch.plug.dll`
- `hack/scripts/*.lua`
- `dfhack-config/DFxTwitch/config.example.json`

## Compatibility

- Dwarf Fortress (Steam) 53.12
- DFHack 53.12-r1
- Windows 10/11
